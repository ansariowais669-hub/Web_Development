# 📝 iTask - ToDo List Web Application

A modern and responsive **ToDo List Web Application** built with **React**, **Vite**, and **Tailwind CSS**. The application allows users to efficiently manage their daily tasks with a clean and intuitive user interface. All tasks are stored in the browser using **Local Storage**, ensuring they persist even after refreshing or reopening the application.

---

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed
* 👀 Show/Hide completed tasks
* 💾 Automatic data persistence using Local Storage
* 📱 Responsive design for desktop and mobile devices
* 🎨 Modern UI built with Tailwind CSS
* 🆔 Unique IDs generated for every task using UUID
